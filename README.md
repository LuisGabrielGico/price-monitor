# Price Monitor

Automation system for collecting prices from the web.

## Technologies
- Python
- Selenium
- Pandas
- PostgreSQL

## Features
- Automatic web scraping
- Database storage
- CSV report generation
- Automation with scheduler

## How to run
1. Create venv
2. Install dependencies
3. Run main.py
4. Run scheduler.py
